# Contribution Policy

## Status: Restricted

This project is **private property** (All Rights Reserved). Contributions are accepted only from pre-approved collaborators who have signed a contribution agreement.

## How to Become a Contributor

1. Contact kiminbean@gmail.com with your intended contribution scope
2. If approved, you will receive a Contributor License Agreement (CLA)
3. Sign and return the CLA before submitting any Pull Request

## Pull Request Requirements

- **CLA Required**: No CLA = no merge. No exceptions.
- **Branch naming**: `feature/<scope>` or `fix/<scope>` only
- **Tests**: All new code must include passing tests
- **License**: All contributions become the property of the copyright holder under the All Rights Reserved license

## Prohibited

- Submitting code without an approved CLA
- Including third-party code without verifying its license compatibility
- Removing or altering copyright notices
- Using AI-generated code without review and attribution

## Code Review

All PRs require at least one approval from the repository owner before merging.
