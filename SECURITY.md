# Security Policy

## Reporting a Vulnerability

This project is private property. If you discover a security vulnerability, please report it **privately**:

- **Email**: kiminbean@gmail.com
- **Do NOT open a public GitHub issue** for security vulnerabilities

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial Assessment**: Within 7 days
- **Fix Release**: Severity-dependent (Critical: 7 days, High: 30 days, Medium/Low: 90 days)

## Scope

This policy covers the latest release branch only. Unsupported versions will not receive security patches.

## Unauthorized Access

Any unauthorized attempt to access, modify, or exploit this software or its infrastructure will be treated as a violation of applicable laws, including but not limited to the Computer Fraud and Abuse Act and relevant copyright legislation.
