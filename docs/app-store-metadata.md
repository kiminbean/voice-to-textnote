## App Description

### 한국어 (Korean)

```
VoiceToTextNote - AI가 만나는 회의의 새로운 경험

🎤 음성 녹음부터 AI 요약까지, 한 번에.
회의, 강의, 인터뷰 등 모든 음성을 녹음하고 AI가 자동으로 텍스트로 변환해 드립니다.

✨ 주요 기능

• 실시간 음성 녹음
  - 백그라운드 녹음 지원으로 앱을 나가도 계속 녹음
  - 고품질 오디오 캡처

• AI 텍스트 변환 (STT)
  - 정확한 음성-텍스트 자동 변환
  - 화자 분리(Diarization)로 누가 말했는지 식별
  - 다국어 지원

• 스마트 회의 요약
  - AI가 핵심 내용을 자동 요약
  - 실행 항목(Action Item) 자동 추출
  - 키워드 및 주제 자동 분류

• 팀 협업
  - 팀 스페이스에서 회의록 공유
  - 멤버 관리 및 권한 설정

• 검색 및 관리
  - 전문 검색으로 필요한 회의 내용 즉시 발견
  - 화자별, 날짜별, 키워드별 필터링
  - 즐겨찾기 및 북마크

• 다양한 내보내기
  - PDF, TXT, DOCX 형식으로 내보내기
  - 공유 기능으로 간편 전달

🔒 보안
- 모든 데이터는 암호화되어 안전하게 저장
- 개인정보 보호 정책 준수
- 소셜 로그인(Google, Apple) 지원

📱 알림
- 변환 완료 시 푸시 알림
- 바로가기 딥링크로 결과 즉시 확인
```

### English

```
VoiceToTextNote - Where AI Meets Your Meetings

🎤 From recording to AI summary, all in one.
Record any meeting, lecture, or interview and let AI automatically transcribe it to text.

✨ Key Features

• Real-time Audio Recording
  - Background recording support - continues even when you leave the app
  - High-quality audio capture

• AI Speech-to-Text (STT)
  - Accurate automatic speech-to-text conversion
  - Speaker diarization to identify who said what
  - Multi-language support

• Smart Meeting Summaries
  - AI automatically generates concise summaries
  - Automatic action item extraction
  - Keyword and topic classification

• Team Collaboration
  - Share meeting notes in team spaces
  - Member management and permission settings

• Search & Organization
  - Full-text search to instantly find meeting content
  - Filter by speaker, date, or keyword
  - Bookmarks and favorites

• Export Options
  - Export to PDF, TXT, DOCX formats
  - Easy sharing with built-in share functionality

🔒 Security
- All data encrypted and securely stored
- Privacy policy compliant
- Social login support (Google, Apple)

📱 Notifications
- Push notifications when processing is complete
- Deep links to instantly view results
```

## App Store Connect Setup Checklist

### Before Submission
- [ ] Apple Developer Program 가입 ($99/year)
- [ ] App Store Connect에서 앱 생성
- [ ] 번들 ID (`com.voicetextnote.app`) 등록
- [ ] 앱 아이콘 (1024x1024) 업로드
- [ ] 스크린샷 6장 이상 (기기별)
- [ ] 앱 설명 (한국어 + 영어)
- [ ] 키워드 (한국어 + 영어)
- [ ] 지원 URL (privacy policy page)
- [ ] 마케팅 URL (optional)
- [ ] 앱 검토 정보 작성

### Technical Requirements
- [ ] Xcode에서 Archive 빌드
- [ ] 코드 서명 (Distribution 인증서)
- [ ] Provisioning Profile (App Store)
- [ ] TestFlight 베타 테스트 (권장)
- [ ] 앱 전송 (Transporter 또는 Xcode)
- [ ] Strict release readiness 통과:
  `python3 client/scripts/verify_release_readiness.py --strict`

### Post-Submission
- [ ] Apple 리뷰 대기 (1-3일)
- [ ] 리뷰 피드백 대응 (필요 시)
- [ ] 출시 승인 후 배포
